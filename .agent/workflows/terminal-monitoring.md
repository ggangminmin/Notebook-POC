---
description: 실시간 터미널 화면 공유 및 모니터링 워크플로우
---

이 워크플로우는 실행 중인 터미널의 상태를 실시간으로 모니터링하고 사용자에게 보고하는 절차를 정의합니다.

// turbo
1. `run_command`로 실행한 배경 작업의 경우 `command_status`를 사용하여 최신 출력을 확인합니다.
2. 인터랙티브 터미널의 경우 `read_terminal`을 사용하여 현재 화면 내용을 읽어옵니다.
3. 읽어온 내용 중 중요한 상태 변화(에러, 컴파일 완료, 포트 번호 등)를 감지합니다.
4. 요약된 터미널 상태를 사용자에게 정기적으로 공유하여 가시성을 확보합니다.
