# AI 오케스트레이션 워크플로우 (AI Orchestration Workflow)

이 워크플로우는 프로젝트의 효율적인 진행을 위해 각 AI 모델의 강점을 극대화하는 협업 체계를 정의합니다.

## 1. 역할 분담 (Role Definition)

*   **🕹️ Antigravity (최종 지휘)**: 프로젝트의 전체적인 방향성을 설정하고, 각 모델의 결과물을 최종 검토 및 통합합니다. USER와의 소통을 전담하며 전체 오케스트레이션을 리드합니다.
*   **🧠 Gemini (기획)**: 새로운 기능의 메커니즘 설계, 사용자 경험(UX) 기획, 데이터 구조 및 알고리즘의 전략적 설계를 담당합니다.
*   **✍️ GPT (글 정리 및 다듬기)**: MCP(Model Context Protocol)가 완벽하게 연동된 **GPT-5.2** 모델을 사용하여 프로젝트의 문서화, 텍스트 가독성 개선, 그리고 복잡한 로직의 언어적 정제를 수행합니다.
*   **🛠️ Claude (개발)**: 핵심 비즈니스 로직 구현, 프론트엔드/백엔드 아키텍처 설계 및 대규모 코드 작성을 담당합니다.
*   **📊 Codex (개발 및 대시보드)**: 최신 **Codex 5.2** 모델을 사용하여 UI/UX 컴포넌트 구현, 대시보드 시각화, 기술적인 세부 사항 최적화 및 코드 스니펫 생성을 담당합니다.

## 2. 작업 프로세스 (Step-by-Step)

1.  **[Strategy]** Antigravity가 태스크를 수신하면 Gemini에게 기획 및 구조 설계를 요청합니다.
2.  **[Polishing]** 기획안이 나오면 GPT가 MCP를 활용해 요구사항을 명확한 기술 문서로 다듬습니다.
3.  **[Development]** 
    *   Claude가 핵심 기능을 개발합니다.
    *   Codex가 대시보드 및 UI 요소를 구현합니다.
4.  **[Validation]** Antigravity가 최종 코드를 검토하고 `npm run dev` 등을 통해 로컬 서버에서 검증합니다.
5.  **[Delivery]** 최종 결과물을 USER에게 보고합니다.

## 3. 실행 지침
모든 협업은 Antigravity의 지휘 하에 순차적 또는 병렬적으로 진행되며, 각 모델의 전문성을 바탕으로 최고의 품질을 유지합니다.
